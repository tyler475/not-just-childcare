# Not Just Child Care

A one-page funnel for **"Not Just Child Care"** — the free quick-start guide to building a
world-class kids ministry. By Tyler Harris, Ministry Solutions.

Static site, hosted on **GitHub Pages**, served at **https://notjustchildcare.com**.

## Files
- `index.html` — the funnel page (deep green + gold brand)
- `title.jpg` — the hero title graphic
- `CNAME` — custom domain for GitHub Pages (`notjustchildcare.com`)
- `not-just-childcare.pdf` — **the guide itself (add this file).** The "Get the Free Guide"
  buttons link to it as a download.

## To finish it
1. **Drop `not-just-childcare.pdf` into this folder**, then commit + push — the download buttons
   will work immediately.
2. (Optional) **Capture emails instead of a direct download.** Right now the buttons download the
   PDF. To turn this into a lead-capture funnel, swap the button for an email form (ConvertKit,
   Mailchimp, or Formspree) that delivers the guide. Easy to add.

## Editing
Everything is in `index.html` — headlines, the three "what's inside" cards, and the footer are
plain text near the bottom of the file. Edit, then:

```bash
git add -A && git commit -m "Update funnel copy" && git push
```

GitHub Pages redeploys automatically within a minute or two.
